不同视频网站的快进/快退规则各不相同：有的每次跳转 5 秒，有的跳转 10 秒，还有许多网站根本不支持键盘控制。QuickSeek for Videos 为所有视频网站带来统一且可自定义的跳转体验，安装后只需按方向键（← →）即可按照你预设的时长快速跳转。

## 功能

- 支持所有含 `<video>` 标签的网站
- 方向键（← →）控制快进 / 快退
- 跳转时长 1 ～ 60 秒自定义，支持滑块、预设按钮（2s / 5s / 10s / 15s / 30s）、直接输入数字
- 跳转时显示秒数浮层，可关闭
- 设置实时生效，无需刷新页面

## 安装

**Chrome 应用商店（推荐）** — 搜索 **QuickSeek for Videos** 安装。

**手动加载：**

1. 下载并解压本仓库
2. 打开 `chrome://extensions`（Edge 为 `edge://extensions`）
3. 开启右上角「**开发者模式**」
4. 点击「**加载已解压的扩展程序**」，选择项目文件夹

## 使用

1. 打开任意视频页面
2. 点击工具栏图标调整跳转秒数
3. 直接按 **← →** 键跳转

> 焦点在输入框内时方向键不会触发跳转。
