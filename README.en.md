Different video platforms handle seeking differently — some jump 5 seconds at a time, others 10, and many don't support keyboard controls at all. QuickSeek for Videos brings a consistent, customizable seeking experience to every video site. Simply press the arrow keys (← →) to skip forward or backward by your preferred duration.

## Features

- Works on any site with an HTML5 `<video>` element
- Arrow keys (← →) to skip forward or backward
- Customize skip duration from 1 to 60 seconds — via slider, preset buttons (2s / 5s / 10s / 15s / 30s), or direct input
- On-screen overlay shows the skip amount; can be turned off
- Settings take effect immediately — no page refresh required

## Installation

**Chrome Web Store (Recommended)** — Search for **QuickSeek for Videos**.

**Load Unpacked:**

1. Download and unzip this repo
2. Open `chrome://extensions` (Edge: `edge://extensions`)
3. Enable **Developer mode** in the top-right corner
4. Click **Load unpacked** and select the project folder

## Usage

1. Open any video page
2. Click the toolbar icon to set your skip duration
3. Press **← →** to seek

> Arrow keys won't trigger seeking when an input field is focused.
